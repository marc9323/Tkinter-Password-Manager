Tkinter Password Manager

CREATE, READ, UPDATE, AND DELETE WEB ACCOUNT DATA:

Never forget your password again!



Tkinter Password Manager Application Help:

  • This application allows multiple users to enter and store information pertaining to web accounts.
  • To register, use the 'Registration' box. Please use an email address for your username and select a password.
  • You must use an email address as your username
  • To login, enter your username(email) and password.
  • For convenience you can save your login so the app will open up to your account by checking the 'Save Login' box.
  • When entering web account data please use a unique name for each account. i.e. for two Gmail accounts use 'Gmail I', 'Gmail II', etc.
  • Hit 'Print Report' to output all of your web account data to a text file.
  • The text file, password-report.txt, will be generated in the application directory.

Web Account Field Description
Account Name Name for the account, i.e. Fifth Third Bank, Gmail, etc. Name must be unique.
Account Link URL, you can copy/paste this in, i.e. http://www.gmail.com
Account Username Your username for the account
Account Email Email associated with the account, i.e. throwaway email used for secondary facebook account
Password Password for the account
Security Questions Answers to security questions, if any, i.e. France, 1942, Smith
Note Any notes regarding the account, i.e. contact name, billing dates, etc.
